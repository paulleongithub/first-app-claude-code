# First app claude code

Projeto iniciado com backup automático via Git/GitHub.

## Lista de Compras — Setup

1. Crie um projeto em [supabase.com](https://supabase.com).
2. No SQL Editor do projeto, rode o script `supabase/migrations/0001_create_items.sql`.
3. Copie `.env.example` para `.env.local` e preencha `VITE_SUPABASE_URL` e `VITE_SUPABASE_ANON_KEY` (em Project Settings → API).
4. Instale as dependências e rode o app:

   ```
   npm install
   npm run dev
   ```
